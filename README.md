# Student Event Registration System

## Project Overview
This project is a lightweight web application built using PHP and MySQL following the 3-tier web architecture concept.

The application allows users to register for events online. It is designed for Raspberry Pi Zero 2W, therefore the system is simple and lightweight.

## Features
- View available events
- Register for events
- Store participant data in database
- Admin can manage event list

## Technologies
- PHP
- MySQL
- HTML/CSS
- Apache

## 3-Tier Architecture

### Presentation Layer
Frontend interface using HTML/CSS.

### Application Layer
PHP processes user requests and business logic.

### Data Layer
MySQL database stores events and participant information.
